# 🔬 Experimento: Algoritmos de Busca em Rust

Projeto desenvolvido para comparar algoritmos de busca sequencial.

## 📌 Algoritmos Implementados

- Busca Sequencial Simples
- Busca Sequencial com Interrupção Antecipada

## 🧪 Exercícios Implementados

- Busca de Strings
- Contagem de Ocorrências
- Retorno de todas as posições encontradas

## 🚀 Como Executar

```bash
cargo run --release
```

## 📊 Objetivo

Demonstrar empiricamente:

- Diferença de eficiência entre algoritmos
- Impacto do tamanho da entrada
- Conceito de custo computacional (O(n))
